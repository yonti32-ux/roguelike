Sprite Directory Structure

Place your sprite files in the appropriate category folder:

- entity/     : Player, enemies, NPCs, chests, merchants
- tile/       : Floor tiles, walls, stairs
- item/       : Weapons, armor, consumables
- battle/     : Battle-specific sprites
- ui/         : UI elements and icons
- effect/     : Status effect icons
- skill/      : Skill icons
- terrain/    : Battle terrain sprites

File naming:
- Basic: {sprite_id}.png
- Variant: {sprite_id}_{variant}.png

Example files to add:
- sprites/entity/player.png
- sprites/entity/enemy_default.png
- sprites/tile/floor.png
- sprites/tile/wall.png
- sprites/item/sword.png

See SPRITE_SYSTEM.md for detailed documentation.
