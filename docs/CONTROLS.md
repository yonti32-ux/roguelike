## General Controls

These controls work across all game modes:

| Key | Action | Description |
|-----|--------|-------------|
| **ESC** | Cancel/Close | Close current overlay or screen |
| **X** | Cancel/Close | Close current overlay or screen |
| **I** | Inventory | Open/close inventory screen |
| **C** | Character Sheet | Open/close character sheet |
| **J** | Quest Screen | Open/close quest/journal screen |
| **T** | Skill Screen | Open/close skill tree screen (exploration mode) |
| **TAB** | Cycle Screens | Cycle through open screens (when in shop) |
| **Shift+TAB** | Toggle Buy/Sell | Toggle between buy/sell modes in shop |

## Overworld Mode

| Key | Action | Description |
|-----|--------|-------------|
| **W / ↑** | Move Up | Move player one tile north |
| **S / ↓** | Move Down | Move player one tile south |
| **A / ←** | Move Left | Move player one tile west |

| **E** | Move NE | Move diagonally northeast |
| **Z** | Move SW | Move diagonally southwest |
| **C** | Move SE | Move diagonally southeast |

## Battle Mode

Turn-based combat controls:

| Key | Action | Description |
|-----|--------|-------------|
| **W / ↑** | Move Up | Move active unit one tile up |
| **S / ↓** | Move Down | Move active unit one tile down |
| **A / ←** | Move Left | Move active unit one tile left |
| **D / →** | Move Right | Move active unit one tile right |
| **Mouse** | Movement | Click to move to tile (pathfinding) |
| **SPACE** | Basic Attack | Attack adjacent enemy with basic attack |
| **Q** | Skill 1 | Use skill in slot 1 |
| **E** | Skill 2 | Use skill in slot 2 |
| **F** | Skill 3 | Use skill in slot 3 |
| **R** | Skill 4 | Use skill in slot 4 |
| **G** | Guard | Use guard skill (if available) |
| **TAB** | End Turn | End current unit's turn |
| **L** | Battle Log | Toggle battle message log |
| **I** | Inventory | Open/close inventory screen |
| **C** | Character Sheet | Open/close character sheet |

## UI Screens

Controls when viewing inventory, character sheet, shop, etc.:

| Key | Action | Description |
|-----|--------|-------------|
| **W / ↑ / K** | Scroll Up | Scroll list up |
| **S / ↓ / J** | Scroll Down | Scroll list down |
| **Page Up** | Page Up | Scroll one page up |
| **Page Down** | Page Down | Scroll one page down |
| **Enter / Space** | Confirm | Select/use/equip selected item |
| **ESC / X** | Cancel | Close screen or cancel action |
| **TAB** | Next Screen | Cycle to next screen (in shop) |
| **I** | Inventory | Jump to inventory screen |
| **C** | Character | Jump to character sheet screen |

### Inventory Screen Specific

| Key | Action | Description |
|-----|--------|-------------|
| **F5** | Filter: All | Show all items |
| **F6** | Filter: Equipped | Show only equipped items |
| **F7** | Filter: Unequipped | Show only unequipped items |
| **Ctrl+S** | Sort | Cycle through sort modes (Name, Rarity, Attack, Defense, HP) |
| **Ctrl+F** | Search | Toggle search mode (type to filter items) |
| **Ctrl+R** | Reset | Clear all filters/sorts/search |

## Cheat/Debug Mode

**Note:** Cheat mode must be enabled with **F9** before using cheats (except F10).

### Toggle Cheat Mode

| Key | Action | Description |
|-----|--------|-------------|
| **F9** | Toggle Cheat Mode | Enable/disable cheat mode |
| **F10** | Debug Sprites | Toggle debug sprite overlay (no cheat mode needed) |

### Universal Cheats (All Modes)

| Key | Action | Description |
|-----|--------|-------------|
| **F2** | Heal Player | Fully heal the player character |
| **F3** | Add Gold | Add 100 gold |
| **F4** | Add XP | Add 25 experience points |

### Overworld Mode Cheats

| Key | Action | Description |
|-----|--------|-------------|
| **F1** | Reveal Map | Reveal entire overworld map (explore all tiles) |
| **F5** | Teleport to POI | Teleport to a random Point of Interest |
| **F6** | Teleport to Center | Teleport to center of overworld map |

### Exploration Mode Cheats

| Key | Action | Description |
|-----|--------|-------------|
| **F1** | Reveal Map | Toggle full map reveal (ignore FOV radius) |
| **F8** | Skip Floor | Skip to next floor |

### Battle Mode Cheats

| Key | Action | Description |
|-----|--------|-------------|
| **F1** | Kill Enemies | Kill all enemy units |
| **F2** | Heal Players | Fully heal all player units |
| **F5** | Heal All | Fully heal all units (player + enemies) |
| **F6** | Refill Resources | Refill stamina/mana/movement for player units |
| **F7** | Skip Turn | Skip current unit's turn |
| **F8** | Win Battle | Defeat all enemies and win battle |

## Notes

- **Cheat Mode**: Most cheats require cheat mode to be enabled (F9). F10 (debug sprites) and F12 (debug console) work without cheat mode.
- **Key Conflicts**: Some keys have different meanings in different modes (e.g., F1 in overworld vs exploration vs battle).
- **Modifier Keys**: Some inventory functions use Ctrl+key combinations.
- **Mouse Controls**: Battle mode supports mouse for movement and targeting.
- **Screen Navigation**: When multiple screens are open (e.g., in shop), TAB cycles between them.
